# Taiwan Surf Trip

Australia-first validation website for a Taiwan surf trip matching service.

## Positioning

Taiwan Surf Trip helps Australian travellers choose the right Taiwan surf area, camp, hostel, lesson or local guide without digging through fragmented local websites.

## SEO focus

Primary keywords:

- Taiwan surf trip
- Taiwan surf camp
- surfing Taiwan from Australia
- Taiwan surf guide
- Dulan surf hostel
- Taitung surf camp
- uncrowded surf Asia

## Local development

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.
